# 💬 DataSense Pro 企业即时通讯系统

## 🎯 项目概述
基于OpenIM SDK的企业级即时通讯系统，提供完整的聊天、群组管理、文件传输等功能。

## ✅ 核心功能
- **即时通讯**：单聊、群聊、消息类型丰富
- **群组管理**：创建群组、成员管理、权限控制
- **消息功能**：发送、撤回、已读状态、搜索
- **会话管理**：会话列表、置顶、免打扰、草稿
- **文件传输**：多格式文件上传下载
- **实时通知**：WebSocket实时消息推送

## 🛠️ 技术栈
**前端**：Vue3 + Element Plus + WebSocket
**后端**：Spring Boot + MyBatis-Plus + MySQL
**通讯**：OpenIM SDK + WebSocket
